# Minimal tileset viewer 

The goal is to provide a fast way to visualize a tileset. It is based on 3d-tileset-viewer. 

## How to install
 ```
 git clone https://github.com/charly-perspectives/3D-tiles-viewer.git
 cd 3D-tiles-viewer
 npm install
 ```

## How to use 

### set up local server
```
cd ..
npx http-server
```
